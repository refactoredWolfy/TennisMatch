# 🎾 Tennis Game Simulator (C#)

A simple console-based tennis game simulator written in C#.  
This project simulates a tennis game between two players using real tennis scoring rules.

---

## 🚀 Features

- 🎯 Realistic tennis scoring system:
  - 0 → 15 → 30 → 40
  - Deuce
  - Advantage
  - Game win logic
- 👥 Two-player system (Player A vs Player B)
- 🎮 Interactive gameplay:
  - Choose who wins each point (A / B)
  - Or let the program decide randomly
- 🔁 Replay option after each game
- 📊 Displays final statistics (points played, final score)

---

## 🧠 What I practiced in this project

- Object-oriented programming (classes, encapsulation)
- Method design and logic structuring
- Input validation
- Control flow (loops, conditions)
- Working with strings and formatting
- Simple game logic implementation

---

## 🏗️ Project Structure

- `TennisMatch` class  
  Handles:
  - player names
  - scoring logic
  - game state (deuce, advantage, win)

- `Program` class  
  Handles:
  - user input
  - game loop
  - interaction and output
